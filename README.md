# Digital Evidence - Financial Transaction Verification App

## Project Overview

**Digital Evidence** is a secure, multi-party financial transaction verification application.

## Tech Stack

- Frontend: React 18, Vite, TensorFlow.js
- Backend: Node.js, Express, MongoDB
- Authentication: JWT
- Face Recognition: face-api.js

## Quick Start

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend
cd frontend && npm install && npm run dev
```
